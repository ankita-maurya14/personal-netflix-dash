# Personal Netflix Dashboard (Power BI)

## Project Overview

The Personal Netflix Dashboard is an interactive Power BI report
designed to analyze Netflix content based on type, genre, release year,
country, rating, and seasons.

This dashboard helps users explore content distribution trends and gain
insights into movies and TV series performance across different
dimensions.

------------------------------------------------------------------------

## Key KPIs

-   Total Titles: 21
-   Total Genres: 21
-   Total Countries: 4

------------------------------------------------------------------------

## Dashboard Features

### 1. Content Type Distribution

-   Pie chart comparing:
    -   Movies
    -   TV Series

### 2. Title by Country

-   Bar chart showing number of titles by:
    -   United States
    -   India
    -   Japan
    -   Korea

### 3. Title by Rating

-   Visual breakdown by rating categories:
    -   U/A 13+
    -   U/A 16+
    -   A
    -   (Blank)

### 4. Title by Year

-   Horizontal bar chart displaying titles based on release year.

### 5. Title by Seasons

-   Bar chart showing number of seasons for TV series.

------------------------------------------------------------------------

## Filters & Slicers

The dashboard includes interactive slicers for:

-   Title
-   Release Year
-   Movie or TV Series
-   Seasons
-   Genre
-   Type

Users can dynamically filter visuals to explore specific content.

------------------------------------------------------------------------

## Tools & Technologies

-   Power BI Desktop
-   DAX (Data Analysis Expressions)
-   Power Query
-   Excel Dataset

------------------------------------------------------------------------

## Insights Derived

-   Movies slightly outnumber TV series in the dataset.
-   The United States and India contribute a major portion of titles.
-   U/A 16+ rating appears frequently among titles.
-   TV series vary significantly in number of seasons.

------------------------------------------------------------------------

## How to Use

1.  Open the `.pbix` file in Power BI Desktop.
2.  Use slicers to filter by genre, year, or type.
3.  Click on visuals to enable cross-filtering.
4.  Analyze trends by country, rating, and seasons.
